Instalador para Windows

La descarga se toma automáticamente del último release publicado en GitHub. No requiere cuenta de GitHub.

Aviso de Windows SmartScreen: si aparece “Windows protegió su PC”, seleccione Más información y luego Ejecutar de todas formas. El aviso puede aparecer mientras el instalador no cuente con firma digital de editor.
Requisitos: Windows 10 o superior, permisos de usuario estándar e internet sólo para descarga y actualizaciones.

Flujo básico de trabajo

1

Instale SIEOP Desktop

Descargue el instalador desde esta página y ejecútelo en su equipo. La aplicación se instalará para el usuario actual.

2

Abra el proyecto

Use el archivo .sieopproj entregado por Obras Públicas. Ese archivo contiene los datos base de la obra.

3

Genere el paquete

Al terminar la estimación, use la opción de envío al portal. SIEOP guardará un paquete .sieopx en la carpeta del proyecto.

Preguntas frecuentes

¿Debo crear el proyecto manualmente?

No. Para obras asignadas por la Dirección de Obras Públicas, debe abrir el archivo .sieopproj que se le entregue. Ese archivo ya contiene los datos base de la obra.

¿Qué archivo debo enviar a Obras Públicas?

Debe enviar el paquete .sieopx generado por SIEOP Desktop. No envíe la carpeta completa del proyecto salvo que se le solicite expresamente.

¿Puedo usar SIEOP Desktop sin internet?

Sí. La captura normal de estimaciones es local. Internet se utiliza para descargar el instalador y verificar actualizaciones.

¿Qué pasa si Windows muestra “Editor desconocido”?

Es una advertencia normal mientras el instalador no esté firmado digitalmente con certificado de editor. Seleccione Más información y luego Ejecutar de todas formas.
